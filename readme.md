# presets

|name|describe|scene|
|---|---|---|
|abs|absolute mode presets|osu!std, blender, etc.|
|art|artist mode presets|krita, blender, etc.|
|mouse|relative mode presets|mc, ttf2, cs2, etc.|

# quick switch presets

> Provides a command `t`, equivalent to `otd loadsettings $presets`
> change presets like

```shell
t abs
t art
t mouse
```

# see also
> Run otd-daemon as admin for better experience on windows(may use msys2)
